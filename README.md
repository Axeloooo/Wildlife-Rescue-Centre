### How to run

- Compile

```bash
javac -cp "lib/mysql-connector-java-8.0.23.jar" -d bin src/edu/ucalgary/oop/*.java
```

- Run in MacOS

```bash
java -cp "lib/mysql-connector-java-8.0.23.jar:bin" edu/ucalgary/oop/Main
```

- Run in Windows

```bash
java -cp "lib/mysql-connector-java-8.0.23.jar;bin" edu/ucalgary/oop/Main
```
