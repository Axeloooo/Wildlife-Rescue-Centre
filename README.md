### How to run

- Compile

```bash
javac -cp "lib/mysql-connector-java-8.0.23.jar" src/Main.java
```

- Run in MacOS

```bash
java -cp "lib/mysql-connector-java-8.0.23.jar:src" Main
```

- Run in Windows

```bash
java -cp "lib/mysql-connector-java-8.0.23.jar;src" Main
```
